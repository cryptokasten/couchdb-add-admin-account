#+TITLE: CouchDB: Add admin account

** System environment

- CentOS Linux release 8.1.1911 (Core)

- couchdb-3.1.0

** Config file location

Default config file location is =/opt/couchdb/etc/local.ini=.

** Raw password

#+BEGIN_SRC conf
[admins]
admin = rawpassword
